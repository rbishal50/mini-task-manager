# Mini Task Manager

#### **A monorepo for web-app and api for Mini Task Manager**

### Quick Run (Development mode)

#### _Run the following commands after cloning the project in root of the project:_

#### npm install

#### npm run dev

### Quick Run (Production mode)

#### _Run the following commands after cloning the project in root of the project:_

#### npm install

#### npm run build

#### npm run start
